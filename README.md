# NeuroChess

resources - ВРЕМЕННО ОГРАНИЧЕННА НА РЕДАКТИРОВАНИЕ И РАСПРОСТРАНЕНИЕ

Спустя примерно год я наконец-то закоммитил одну из сохранённых версий проекта.

Это **не последняя версия**. Репозиторий создан в первую очередь как **резервная выгрузка**, чтобы не потерять весь накопленный прогресс окончательно. Проект долгое время существовал вне нормального git-репозитория, часть материалов была утеряна, поэтому текущее состояние следует воспринимать как **снимок одной из рабочих стадий разработки**, а не как финальный релиз.

## Текущее состояние

На данный момент проект загружен в том виде, в котором его удалось сохранить.

Что важно понимать:

1. **Оригинальная версия ИИ утеряна безвозвратно.**  
   Исходные модели и часть связанных материалов восстановить не удалось. Даже попытки найти или восстановить данные через Hugging Face не помогли.

2. **Ассеты нельзя свободно редактировать или переиспользовать.**  
   Используемые игровые ресурсы распространяются по **платной лицензии**. Право на использование у меня есть, но саму лицензию ещё нужно найти и поднять из архивов. До прояснения этого вопроса стоит считать, что ассеты **не предназначены для стороннего редактирования, копирования и повторного использования**.

3. **Это явно не самая свежая версия проекта.**  
   Репозиторий создан как страховочная публикация, чтобы зафиксировать код, структуру проекта и модели в текущем найденном состоянии. Некоторые функции могут отсутствовать, быть недоделанными или работать не так, как работали в более поздних локальных версиях.

4. **Режим с нейросетью сейчас не работает.**  
   В этом состоянии проекта AI-режим не находится в полностью рабочем виде. Позже постараюсь его восстановить и починить.

5. **Для нейросетевых файлов используется Git LFS.**  
   Большие файлы моделей хранятся через Git Large File Storage. Без Git LFS проект может скачаться некорректно или без нужных весов.

## Особенности проекта

Несмотря на текущее состояние, в проекте уже были реализованы или частично реализованы следующие вещи:

- **ИИ на базе GPT-2**, обученный примерно на **3 миллионах шахматных партий** и дополнительном контенте.  
  Одна из целей была в том, чтобы модель могла работать даже на очень слабом железе, включая устройства уровня **Intel Atom**.

- **Cyber** — режим ИИ, ориентированный на обучение новых игроков.  
  Этот режим задумывался как более дружелюбный и обучающий вариант взаимодействия.

- **Martin** — режим на основе **Minimax**.  
  Более классический шахматный движок без нейросетевой составляющей.

- **Подсказки.**  
  В проекте предусмотрена система подсказок для игрока.

## Важно перед запуском

Если вы клонируете этот репозиторий, используйте:

```bash
git lfs install
git clone <repo_url>
```

Если репозиторий уже клонирован без LFS, может понадобиться:
```bash
git lfs pull
```
